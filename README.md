# Job-Opportunity-Quantitative-Evaluation-Model

### 第一阶段：因子重构与打分规则（数据字典）

所有子项目统一采用 **0.0 ~ 1.0** 的标准化得分（1.0为最优），便于跨维度计算。

#### A. 地理与生存成本因子（权重：15%）

| 新子因子 | 量化公式/打分规则 | 数据来源 |
| :--- | :--- | :--- |
| **A1 税后购买力指数** | **（年总包税后 - 当地年均基本生活开支）/ 年总包税后**。比值 >0.6 得1.0，0.4-0.6得0.6，<0.4得0.3。 |  Numbeo / 各地统计局 + 个税计算器 |
| **A2 灵活办公自由度** | 每年允许远程办公的天数。 >120天=1.0， 60-120天=0.7， <60天=0.4， 全勤=0.2。 | Offer条款 / HR面询 |
| **A3 强制调动风险（惩罚项）** | 合同含“全球/全国调配”条款且无否决权，此项直接**乘以惩罚系数0.5**；有协商权则系数0.9；无条款则为1.0（满分）。 | 劳动合同 |
| **A4 城市通勤效率** | 单程通勤时间（分钟）。 <30min=1.0， 30-60=0.6， 60-90=0.3， >90=0.1。 | 高德/谷歌地图API |

---

#### B. 平台硬实力因子（权重：30%）

| 新子因子 | 量化公式/打分规则 | 数据来源 |
| :--- | :--- | :--- |
| **B1 行业相对排名** | 所在赛道（细分领域）的前5名分位。Top 1%=1.0，Top 5%=0.8，Top 20%=0.5，Top 50%=0.3。 | Gartner / IDC / QS学科排名 |
| **B2 营收增速（3年CAGR）** | >30%=1.0， 15%-30%=0.7， 0%-15%=0.5， 负增长=0.2。 | 公司年报 / 企查查 / 招股书 |
| **B3 人才资本密度** | （硕博占比 + 高级职称/专家占比）/2。 >70%=1.0， 50%-70%=0.7， <50%=0.4。 | LinkedIn / 官网团队介绍 |
| **B4 国际化实质指数** | 海外营收占比或非本国籍员工占比。 >40%=1.0， 20-40%=0.6， <20%=0.3。 | 公司年报 / 行业报告 |
| **B5 组织稳定性（离职率）** | 近一年主动离职率。 <5%=1.0， 5-15%=0.7， 15-30%=0.4， >30%=0.1。 | 脉脉 /  Glassdoor / 猎头背调 |

---

#### C. 职业资产增值因子（权重：25%）

| 新子因子 | 量化公式/打分规则 | 数据来源 |
| :--- | :--- | :--- |
| **C1 技能护城河深度** | 掌握该技能所需平均年限（行业基准）。 >5年=1.0， 3-5年=0.7， 1-3年=0.4， <1年（通用技能）=0.2。 | 行业薪酬报告 / O*NET数据库 |
| **C2 技能可迁移广度** | 该岗位技能可适用的目标行业数量。 >5个行业=1.0， 3-4个=0.7， 1-2个=0.4， 仅限本司=0.1。 | 招聘网站（搜索该技能匹配的岗位大类） |
| **C3 行业生命周期位置** | 成长期（渗透率<15%）=1.0， 成熟期（渗透率40-70%）=0.7， 衰退期=0.2。 | 艾瑞咨询 / 国家统计局行业增速 |
| **C4 知识体系学术溢价** | 该岗位是否鼓励发表专利/论文，或与高校联合培养。是=1.0， 内部技术论坛=0.6， 严禁对外分享=0.2。 | 岗位JD / 部门老板面试透露 |
| **C5 跳槽溢价系数** | 该平台离职员工跳槽后平均薪资涨幅。 >30%=1.0， 15-30%=0.7， <15%=0.4。 | 脉脉“薪酬查询” / 猎头反馈 |

---

#### D. 生存与生活质量因子（权重：20%）

| 新子因子 | 量化公式/打分规则 | 数据来源 |
| :--- | :--- | :--- |
| **D1 实际时薪水平** | **（年总包 / 12） / （每周工时 * 4）**。取当地行业25分位数对比： >行业P75=1.0， P50-P75=0.7， <P50=0.4。 | 自己计算 vs 行业薪酬报告 |
| **D2 带薪休息总量** | 年假天数 + 法定节假日 + 寒暑假（仅限学校）+ 双休周数*2。 >150天=1.0， 120-150=0.7， 90-120=0.4， <90=0.2。 | Offer / 公司福利手册 |
| **D3 加班刚性（惩罚）** | 强制固定加班（如996），此项直接**乘以惩罚系数0.6**；偶尔项目制加班（0.9）；完全双休不加班（1.0）。 | 职场社区（真实口碑） |
| **D4 健康保障覆盖率** | 补充医疗保险+企业年金+体检。全覆盖直系亲属=1.0，仅本人高端险=0.7，仅基础社保=0.3。 | 福利手册 |

#### E. 主观适配与期权因子（权重：10%）

| 新子因子 | 打分规则（1-10分，归一化除以10） | 操作建议 |
| :--- | :--- | :--- |
| **E1 内容兴趣强度** | 问自己：即使不给钱，我是否愿意每周花10小时研究这个？非常愿意=10，完全不愿=1。 | 自我内省 + 职业测评（霍兰德） |
| **E2 团队化学反应（面试感知）** | 面试中主动提问的次数 / 面试官微笑回应的次数。比值高则分高。 | 面试备忘录记录 |
| **E3 长期愿景一致性** | 该岗位经历是否是你5年后理想履历中的“必备一环”？是=10，否=1。 | 5年职业规划倒推 |

### 第二阶段：权重分配与最终评分公式

根据**决策理智程度（理性权重 > 感性权重）**，建议采用以下锚定权重。你也可根据自身是“风险厌恶型”还是“进取型”微调（±5%）。

| 维度 | 符号 | 建议权重 |
| :--- | :--- | :--- |
| 地理与生存成本 | $W_A$ | **15%** |
| 平台硬实力 | $W_B$ | **30%** （职业生涯前10年权重最高） |
| 职业资产增值 | $W_C$ | **25%** （对抗AI时代最关键） |
| 生活质量 | $W_D$ | **20%** （决定可持续性） |
| 主观适配 | $W_E$ | **10%** （直觉往往是潜意识的精确计算） |

---

**最终综合评分公式（满分1.0，换算为百分制乘以100）：**

$$ Score = \sum (W_i \times S_i) $$

展开为：
$$ Score = 0.15 \times S_A + 0.30 \times S_B + 0.25 \times S_C + 0.20 \times S_D + 0.10 \times S_E $$

其中，**各维度子得分计算公式**（采用加权平均，权重相等子项均为1/N，但**惩罚项为连乘系数**）：

$$ S_A = \frac{A1 + A2 + A4}{3} \times A3 $$

$$ S_B = \frac{B1 + B2 + B3 + B4 + B5}{5} $$

$$ S_C = \frac{C1 + C2 + C3 + C4 + C5}{5} $$

$$ S_D = \frac{D1 + D2 + D4}{3} \times D3 $$

$$ S_E = \frac{E1 + E2 + E3}{3} $$

### 第三阶段：实际应用指南（如何统一数据格式）

为了便于比较多个Offer，请制作一张 **Excel决策矩阵表**，统一按以下格式填入：

| 维度 | 子因子 | Offer A 原始数据 | Offer A 标准化得分 (0-1) | Offer B 标准化得分 |
| :--- | :--- | :--- | :--- | :--- |
| **A** | A1 购买力 | 税后35万，开销20万 → (35-20)/35=0.43 | **0.6** | ... |
| | A2 远程办公 | 每年52天 | **0.7** | ... |
| | A3 强制调动 | 无此条款 | **1.0** | ... |
| | A4 通勤 | 45分钟 | **0.6** | ... |
| *维度A小计* | *$S_A$* | *——* | *(0.6+0.7+0.6)/3 * 1.0 = **0.63*** | *...* |

### 最终决策建议（模型背后的理智逻辑）

1. **门槛原则**：如果 **D3（加班刚性）≤ 0.6** 且 **A3（强制调动）≤ 0.5**，无论总分多高，建议直接淘汰（说明该工作侵蚀基本人权或毫无稳定预期）。
2. **权重动态调整**：若你是**35岁以上**，建议手动将 $W_D$（生活质量）调至30%，$W_B$（平台）调至20%。
3. **分数差值阈值**：两个Offer分差 **> 0.15**（即15分），优先选高分；分差在 **0.05 - 0.15** 之间，属于“模糊地带”，此时建议选择 $S_C$（职业资产增值）得分更高的那个，因为这是时间不可逆的复利。
