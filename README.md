#### 维度 A：地理位置与生存成本（权重15%）

| **A2** | 灵活变更城市自由度 | 0~1 数值 | **线性**：\( S=x \)（0=完全固定，1=可全球任意变更） |
| **A3** | 强制调动风险 | 0~1 数值 | **反向线性**：\( S=1-x \)（0=无风险，1=强制全球调配） |

### 第三部分：核心计算公式

#### 1. 子因子有效得分（置信度加权）

\[
S_i^{有效} = S_i \times C_i
\]

#### 2. 各维度内部聚合均值（\( \mu_{dim} \)）

- **维度A**（内部加权 + 惩罚系数）：

\[
\mu_A = \left( 0.45 \cdot S_{A1} + 0.30 \cdot S_{A2} + 0.25 \cdot S_{A4} \right) \times S_{A3}
\]
*（\( S_{A3} \)作为乘数，若A3未知则不缩放）*

- **维度B、C、D**（等权，置信度加权平均）：

\[
\mu_{dim} = \frac{\sum_{i=1}^{n} (S_i \cdot C_i)}{\sum_{i=1}^{n} C_i} \quad (dim = B,C,D)
\]

- **维度E**（内部权重 \( w_i \)）：

\[
\mu_E = \frac{0.30(S_{E1}C_{E1}) + 0.20(S_{E2}C_{E2}) + 0.20(S_{E3}C_{E3}) + 0.30(S_{E4}C_{E4})}{0.30C_{E1} + 0.20C_{E2} + 0.20C_{E3} + 0.30C_{E4}}
\]

- **维度F**（内部权重 \( w_i \)）：

\[
\mu_F = \frac{0.30(S_{F1}C_{F1}) + 0.40(S_{F2}C_{F2}) + 0.30(S_{F3}C_{F3})}{0.30C_{F1} + 0.40C_{F2} + 0.30C_{F3}}
\]

#### 3. 维度内部方差（\( \sigma_{dim}^2 \)）

\[
\sigma_{dim}^2 = \frac{\sum C_i \times (S_i - \mu_{dim})^2}{\sum C_i}
\]
*（当维度内所有数据均未知时，\( \mu_{dim}=0.5, \sigma_{dim}^2=0.1 \)）*

#### 4. 全局稳健得分（Robust Score）

\[
S_{Robust} = 0.15\cdot\mu_A + 0.20\cdot\mu_B + 0.15\cdot\mu_C + 0.32\cdot\mu_D + 0.08\cdot\mu_E + 0.10\cdot\mu_F
\]

#### 5. 全局总不确定性（标准差）

\[
\sigma_{Total} = \sqrt{ \sum_{dim=A}^{F} \left( W_{dim}^2 \times \sigma_{dim}^2 \right) }
\]

#### 6. 信息完整度惩罚（IIP）

> *总因子数 \( N_{Total} = 24 \)（A4 + B4 + C5 + D4 + E4 + F3）*
\[
IIP = 1 - \left( \frac{N_{Unknown}}{24} \times 0.15 \right)
\]
*（缺失比例越高惩罚越重，最高折价92.5%）*

#### 7. 最终推荐指数（百分制）与90%置信区间

\[
Score_{最终} = S_{Robust} \times IIP \times 100
\]
\[
90\%\ CI = \left[ (S_{Robust} \pm 1.645 \cdot \sigma_{Total}) \times IIP \times 100 \right]
\]
*（结果自动截断在 0~100 分区间内）*

### 第四部分：程序操作说明

#### 1. 启动与权重调节
- 运行程序，首先显示**全局权重调节**界面（默认：A=15%，B=20%，C=15%，D=32%，E=8%，F=10%）。
- 输入 `y` 可自定义六大维度的权重百分比（整数），系统自动归一化；**直接按回车则使用默认权重**。

#### 2. 数据录入通用规范
- **普通数值**：直接输入数字（如 `35`、`0.2`、`85`）。
- **选项类**：按提示输入指定字母（如 C3 输入 `U`/`S`/`D`）或数字（如 F2 输入 `1`~`8`）。
- **标记数据未知**：任意输入环节，若该项完全无数据，请输入小写字母 **`u`** 或直接按 **Enter** 留空。系统将自动填充中性得分（0.5）且置信度极低（0.1），并最终在 IIP 中予以惩罚。
- **置信度选择**：每个已知数据录入后，需选择置信度档次：输入 `H`（高）、`M`（中）、`L`（低），**直接回车默认“中”（0.6）**。

#### 3. 特殊因子输入指引（务必注意）

| 因子 | 交互流程 | 关键注意事项 |
| :--- | :--- | :--- |
| **A1（年终结余）** | 先输入 `1`（直接输入余额）或 `2`（分别输入总包与成本）。若选 `2`，程序自动计算 `收入 - 成本`。 | 单位均为**万元**。若输入 `u` 或回车，整体标记未知。 |
| **C3（行业生命周期）** | 输入大写字母：`U`（上行）、`S`（稳定）、`D`（下行）。 | 程序自动映射为 1.0 / 0.7 / 0.3。输入其他字符或 `u` 则视为未知。 |
| **F1（合同期限）** | 输入具体年数。**若为“无固定期限合同”，请务必输入数字 `0`**。 | 输入 `0` 直接得满分 1.0，这是特殊设定。 |
| **F2（单位类型）** | 输入对应数字（1~8）。 | 映射表已内置（公务员→1.0，外包→0.1），输入数字即可，无需自己算分。 |

#### 4. 输出结果解读
程序输出包含四大模块，阅读顺序建议：

1. **维度分解**：查看 A~F 各维度的稳健得分及内部方差。方差越大代表该维度数据不确定性越高（例如全靠猜测）。
2. **信息完整度**：显示 `已知项/总项(24)` 和 `IIP` 惩罚系数。若已知项过少，IIP 会显著打折最终分数。
3. **最终核心指标**：
   - **推荐指数（百分制）**：横向对比多个 Offer 的核心依据。
   - **90% 置信区间**：给出得分波动范围。区间下限（Lower Bound）是最值得关注的稳健值。
4. **智能决策建议**：
   - 区间宽度 > 25 分 → 数据严重不足，强烈建议补充缺失项。
   - 区间宽度 15~25 分 → 存在风险，核实置信度为“低”的关键项。
   - 区间宽度 < 15 分 → 数据质量良好，结果可信。
   - 评级：≥80 为 A 级（强烈推荐）；65~80 为 B 级（推荐）；50~65 为 C 级（及格）；<50 为 D 级（谨慎）。

#### 5. 多 Offer 对比决策金标准
- 将每个 Offer 分别运行，记录**最终推荐指数**和**置信区间下限**。
- **优先选择**：“推荐指数高”且“置信区间下限更高”的那个（代表最坏情况也能接受）。
- 若两个 Offer 分差 < 5 分，建议重点对比 **维度 C（职业前景）** 和 **维度 F（稳定性）** 的单项得分，这两项关乎长期复利和抗风险能力。
